
# Akan Name Generator

Simple web app that generates an Akan (Ghanaian) name from a user's birth date and gender.

## Demo

- Open `index.html` in a browser, or serve the project folder with a static server:

## Features

- Generate Akan name based on weekday of birth.
- Responsive form UI with validation and reset.
- No build step — plain HTML/CSS/JS.

## Usage

1. Open webpage 
2. Enter your birth date and select your gender.
3. Click "Generate Akan Name" — the result appears below the form.

## Files

- `index.html` — main UI and form markup
- `style.css` — styling for the form and layout
- `script.js` — logic that computes the weekday and maps to Akan names
